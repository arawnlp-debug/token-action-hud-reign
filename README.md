# Token Action HUD: Reign

A customized Token Action HUD integration for the Reign VTT system (ORE) in Foundry Virtual Tabletop. 

## Features
* **Custom Layout:** Organizes actions into Stats, Skills, Combat, Techniques, Custom Moves, Spells, and Disciplines.
* **Smart Rollers:** Automatically integrates with Reign's native `CharacterRoller`, `CompanyRoller`, and `ThreatRoller`.
* **Right-Click Equipping:** Quickly equip and unequip weapons and armor directly from the HUD.

## Installation
1. Open Foundry VTT and go to the **Add-on Modules** tab.
2. Click **Install Module**.
3. Paste the following Manifest URL into the field at the bottom:
   *https://github.com/arawnlp-debug/token-action-hud-reign/releases/latest/download/module.json*
4. Click Install.

## Requirements
* Foundry VTT v11+
* Reign System v2.1.1+
* Token Action HUD Core v2.1.1+
